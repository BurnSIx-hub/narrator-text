# 📜 Narrator Text

**EN:** Cinematic narrator captions for Foundry VTT. The GM displays styled text on every player's screen: custom fonts, size, color, outline and shadow, 9 screen positions (or free X/Y placement), fade/typewriter animations, show duration and a resend history.

**Install:** Foundry VTT → Add-on Modules → Install Module →
```
https://github.com/BurnSIx-hub/narrator-text/releases/latest/download/module.json
```

---

## Что это (RU)

ГМ показывает красивые надписи на экранах всех игроков — как титры рассказчика в кино: «Прошло три года…», «Тем временем в столице…».

### Возможности

- 🖋️ **Оформление** — выбор шрифта (включая встроенные декоративные), размер, цвет, прозрачность, тень, обводка, выравнивание
- 📍 **9 позиций на экране** + произвольная (X%/Y%) и ширина текстовой области
- 🎬 **Анимации** — плавное появление/исчезновение, печатная машинка, мгновенно; настраиваемые длительности
- ⏱️ **Время показа** и разрешение накладывать несколько надписей
- 🕘 **История** — повторная отправка прошлых надписей в один клик

### Как пользоваться

1. ГМ открывает «Текст Нарратора» (кнопка на панели инструментов)
2. Вводит текст, настраивает вид и анимацию
3. «Показать» — надпись появляется у всех игроков

### Установка вручную

Скачайте `module.zip` из [последнего релиза](https://github.com/BurnSIx-hub/narrator-text/releases/latest) и распакуйте в `FoundryVTT/Data/modules/narrator-text/`.

Требования: Foundry VTT v13–v14.
